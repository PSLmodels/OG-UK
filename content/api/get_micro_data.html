This function creates an OpenFisca Microsimulation object with the policy specified in reform and the data specified with the data kwarg.

  • year (int) – year of data to simulate

  • reform (OpenFisca Reform object) – IIT policy reform parameters, None if baseline

  • data (DataFrame or str) – DataFrame or path to datafile for the PopulationSim object

rates and other information computed from OpenFisca-UK

oguk.get_micro_data.get_data(baseline=False, start_year=2021, reform=None, data='frs', path='/home/runner/work/OG-UK/OG-UK/oguk', client=None, num_workers=1)[source]¶

This function creates dataframes of micro data with marginal tax rates and information to compute effective tax rates from the PopulationSim object. The resulting dictionary of dataframes is returned and saved to disk in a pickle file.

Parameters

  • baseline (boolean) – True if baseline tax policy

  • start_year (int) – first year of budget window

  • reform (OpenFisca Reform object) – IIT policy reform parameters, None if baseline

  • data (DataFrame or str) – DataFrame or path to datafile for the PopulationSim object

  • path (str) – path to save microdata files to

  • client (Dask Client object) – client for Dask multiprocessing

  • num_workers (int) – number of workers to use for Dask multiprocessing

Returns

dict of Pandas Dataframe, one for each

year from start_year to the maximum year OpenFisca-UK can analyze

OpenFiscaUK_version (str): version of OpenFisca-UK used

Return type
